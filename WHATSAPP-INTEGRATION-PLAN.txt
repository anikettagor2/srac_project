                              6. USAGE EXAMPLES
--------------------------------------------------------------------------------

Fire-and-forget (non-blocking):
```typescript
import { notifyClientProjectCreated, notifyEditorProjectAssigned } from "@/lib/whatsapp";

// Notify client about new project
notifyClientProjectCreated(projectId);

// Notify editor about assignment
notifyEditorProjectAssigned(projectId, editorId, pmName, deadline);
```

With async/await (if you need to handle result):
```typescript
import { notifyClient, notifyPM } from "@/lib/whatsapp";

const result = await notifyClient(projectId, 'client_draft_submitted');
if (!result.success) console.error(result.error);
```

================================================================================
                              HOW IT WILL WORK
================================================================================

EXAMPLE: When editor uploads a draft

1. System detects: "Draft uploaded for project X"
2. System looks up: Client phone number
3. System sends to AiSensy:
   - Campaign: "client"
   - {{1}}: "John"
   - {{2}}: "A new draft is ready for your review! Check your dashboard."
   - {{3}}: "YouTube Video Edit"
4. Client receives WhatsApp message ✅
